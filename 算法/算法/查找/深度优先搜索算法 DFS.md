# 使用场景

一种用于遍历或搜索树或图的算法

# 复杂度

空间复杂度

    迭代：O( 1 )
    递归：O( logn )

时间复杂度

    最优：O( 1 )
    最坏：O( log(n) )
    平均：O( log(n) )

# 过程描述

深度优先遍历的主要思想就是：首先以一个未被访问过的顶点作为起始顶点，沿当前顶点的边走到未访问过的顶点；当没有未访问过的顶点时，则回到上一个顶点，继续试探访问别的顶点，直到所有的顶点都被访问。

沿着某条路径遍历直到末端，然后回溯，再沿着另一条进行同样的遍历，直到所有的顶点都被访问过为止。
